ชุดโครงการชุมชนที่สร้างขึ้นบน shob

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ shob ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-shob](https://github.com/awesome-shob/awesome-shob) และ [shob.cafe](https://shob.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
